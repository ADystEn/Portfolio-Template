1. <navbar> : Eröffnet das Konzept "Navbar"


2. <navbar-structurepanel> : Eingebettes Strukturpanel zur Anordnung der genealogischen Elemente


3.1 <nav class="nav-logo-box"> : Ein Navbar-Element, dass im Strukturpanel angeordnet ist, 1.Level


3.2 <nav class="nav-items-box"> : Ein Navbar-Element, dass im Strukturpanel angeordnet ist, 1.Level

    3.2.1 <div class="nav-item"> : Ein Navbar-Element der navbar-items-box, für direkte Verlinkung auf Klick/Touch 2.level

    3.2.2 <div class="nav-item-dropdown"> : Ein Navbar-Element der navbar-items-box, für ein Dropdown-Menü auf Klick/Touch 2.Level



3.3 <nav class="home-switch"> : Ein Navbar-Element, welches eine Schalter-Funktion hat und zum Seitenstartpunkt führt, 1. Level


3.4 <nav class="burger-off-switch"> : Ein Navbar-Element, welches eine Schalter-Funktion hat und die nav-items-box schließt, 1. Level
