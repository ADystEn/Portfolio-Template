Portfolio-Template

Hier stelle ich diese hier sichtbare Webseite als Template vor.

Die Entscheidung für eine statische Webseite liegt in mehreren Ursachen begründet.

  • 1. Es war eine isolierte Herausforderung die erworbenen Kenntnisse aus HTML, CSS und Javascript Kursen umzusetzen.

  • 2. Insbesondere die Navigation als eine logische Schaltung aus der Elektrotechnik zu begreifen und sie wie eine Software zu programmieren, war dabei eine Hürde fernab der "Tutorial-Hell".

  • 3. Zum gegenwärtigen Zeitpunkt, genügt es dieses Template als Portfolio zu verwenden, dadurch habe ich die kreative Freiheit Dinge auszuprobieren im Bereich des Frontends.
