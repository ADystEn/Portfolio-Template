Portfolio-Template

Hier stelle ich diese hier sichtbare Webseite als Template vor.

Die Entscheidung für eine statische Webseite liegt in mehreren Ursachen begründet.

  • 1. Es war eine isolierte Herausforderung die erworbenen Kenntnisse aus HTML, CSS und Javascript Kursen umzusetzen.

  • 2. Insbesondere die Navigation als eine logische Schaltung aus der Elektrotechnik zu begreifen und sie wie eine Software zu programmieren, war dabei eine Hürde fernab der "Tutorial-Hell".

  • 3. Zum gegenwärtigen Zeitpunkt, genügt es dieses Template als Portfolio zu verwenden, dadurch habe ich die kreative Freiheit Dinge auszuprobieren im Bereich des Frontends.

Yelp-Camp: Full-Stack Web-App

Hier stelle ich das Projekt aus dem Kurs "The Developer Bootcamp 2025" vor.
Das Projekt befindet sich derzeit noch in Überarbeitung. Es handelt sich bei diesem Projekt um eine Fullstack-Web-App, welche mehrere Software- und Webtechnologien kombiniert.
Das Projekt ist hauptsächlich ein sogenanntes "Code-Along" und keine vollständige eigene Schöpfung und ist auf den Autor
des Kurses "Colt Steele" zurückzuführen. Anhand dieses Projekts war es mir möglich tiefer in die Zusammenhänge
der Fullstack Webentwicklung einzutauchen und ich konnte dabei weitreichende Grundkenntnisse zum Thema erfahren.
Viele der enthaltenen Technologien verdienen eine tiefere und eigenständige praktische Ausseinandersetzung um selbstständig und projektbezogen umgesetzt werden zu können. Nur um einige Konzepte, Methoden, Technologien des Kurses und eigene Ergänzungen zu nennen:

Frontend

  • HTML
  • CSS
  • JavaScript

Backend

  • NodeJS
  • NPM
    • Express
    • axios
    • cheerio
    • ejs
    • method-override
    • morgan
    • MongoDB
    • mongoose
